## Overview

This project develops machine learning regression models to predict daily biogas production from a 15-year dataset of a biogas plant in India. The goal is to support operational decision-making and energy supply planning through accurate prediction of daily output.

## Dataset

Available on kaggle: 
<a href="https://www.kaggle.com/datasets/ucimachinelearning/indian-biogas-production-dataset">Biogas Production Dataset</a>
